Bootstrap Responsive website
------------

This website profolio is fully responsive and mobile ready. It has the ability to function faultlessly on all types of browsers and 
modern devices and adapts suitably on small, big or wide screen resolutions. This built is built on top of 
Bootstrap html5 css3 which gave flexibilty to customeise the theme easly as per any perferred User interface requirements. 

Key features
-------------

Twitter Bootstrap 3.3.1
Clean & Developer-friendly HTML5 and CSS3 code
100% Responsive Layout Design
Single Page
Multipurpose theme
Google Fonts Support
Font Awesome 
Smooth Scrolling 
Fully Customizable
Contact Form
